# fastapi_crud

Instagram-style backend built with FastAPI, SQLAlchemy, and SQLite.

Core endpoints:

- `POST /users/`, `GET /users/`, `GET /users/{user_id}`, `PUT /users/{user_id}`, `DELETE /users/{user_id}`
- `POST /posts/`, `GET /posts/`, `GET /posts/{post_id}`, `GET /users/{user_id}/posts`
- `POST /posts/{post_id}/comments`, `GET /posts/{post_id}/comments`
- `POST /posts/{post_id}/likes`, `DELETE /posts/{post_id}/likes`
- `POST /users/{follower_id}/follow/{followed_id}`, `DELETE /users/{follower_id}/follow/{followed_id}`
- `GET /feed/{user_id}`
